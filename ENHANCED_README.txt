🎯 核心功能
================================================================================

1. 矩阵投放系统
   • 一键分配素材到同平台所有账号
   • 支持自定义账号和素材组合
   • 定时投放和批量投放
   • 任务状态跟踪和统计
   • 账号分组管理

2. 代理IP管理
   • IP池管理（增删改查）
   • 自动选择可用代理
   • 定时自动切换IP
   • 使用统计和冷却管理
   • 支持HTTP/HTTPS/SOCKS5

3. 浏览器Cookie管理
   • 统一浏览器配置
   • 自动创建Cookie路径
   • Cookie有效性跟踪
   • 自动刷新提醒

4. 任务调度系统
   • 自动执行矩阵任务
   • 定时IP切换
   • Cookie监控
   • 后台多线程运行

🌟 使用示例
================================================================================

矩阵投放:
  POST /api/enhanced/matrix/tasks/batch-distribute
  {
    "platform_type": 3,
    "file_ids": [1, 2, 3],
    "title": "视频标题"
  }

IP切换:
  POST /api/enhanced/ip-switch/init
  {
    "account_id": 1,
    "switch_interval_minutes": 60
  }

Cookie管理:
  POST /api/enhanced/cookies/auto-create-path
  {
    "account_id": 1,
    "platform_type": 3,
    "username": "user"
  }
